ПРИНЦИПЫ
ДВИЖЕНИЯ

Брутализм не боится анимации — но анимация должна быть ТОЧНОЙ. Не плавной ради плавности — быстрой, резкой, уверенной. Рекомендации из исследований Awwwards 2024.

Тип 1
ВХОД
Появление элементов. Быстро: 0.3–0.5s. Кубическая кривая ease-out. Элемент влетает снизу/сбоку.
Тип 2
HOVER
Мгновенный отклик. 0.15–0.2s. Линейная или ease. box-shadow смещение, translate offset — не масштаб.
Тип 3
ТИКЕР
Бесконечный loop. 10–20s linear. Дублируй контент × 2 для seamless loop без JS. CSS-only.
КИТ · ТЕАТР · КИНО ·   КИТ · ТЕАТР · КИНО ·
Тип 4
STAGGER
animation-delay +0.08s на каждый элемент. Визуальный каскад без JS. Максимум 8–10 элементов.

07

✓ ДЕЛАТЬ

▶
Bebas Neue для всех заголовков. Крупно. Без скруглений. Буквы = архитектура.
▶
border: 3px solid #0A0A0A. Жирные видимые бордеры = структура, которую не прячут.
▶
box-shadow offset (4–8px) без blur. Геометрическая тень = псевдо-3D без иллюзий.
▶
Бегущая строка как полноценный UI-элемент — разделитель секций, носитель информации.
▶
Красный только как акцент. Максимум 15% экрана. Он должен обжигать взгляд.
▶
SVG-анимации: draw, morph, glitch. Честная векторная графика — брутальная и масштабируемая.
▶
Кастомный курсор с mix-blend-mode: multiply. Как на всех топовых Awwwards сайтах.
▶
Огромные фоновые числа (-webkit-text-stroke) как текстурный элемент заднего плана.

✗ НЕ ДЕЛАТЬ

✕
Градиенты как основной приём. Особенно purple-to-pink. Это Anti-brutal по определению.
✕
border-radius больше 0px на ключевых элементах. Никаких «пузырей» и «пилюль».
✕
box-shadow с blur (0px 4px 20px rgba...). Это делает дизайн «воздушным» — против духа.
✕
Inter, Roboto, Arial как основные шрифты. Они нейтральны — а брутализм это всегда позиция.
✕
Hamburger-меню. Брутализм показывает структуру открыто. Навигация = часть дизайна.
✕
Анимации longer than 0.6s для hover. Браузер должен отвечать мгновенно — уверенно, как актёр.
✕
Пастельные цвета без причины. Пергамент (#F0EBE0) — это наш нейтральный. Ниже него не идти.
✕
Карусели и слайдеры. Контент должен быть виден. Прокрутка = пользователь, не автоплей.
